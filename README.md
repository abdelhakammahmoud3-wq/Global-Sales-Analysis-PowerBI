# Global Sales Performance Dashboard
 ## Project Overview
An interactive Power BI dashboard designed to analyze global sales data, providing insights into profit margins, revenue trends, and product category performance across different regions.
 ## Key Features
- **KPI Tracking:** Total Profit, Revenue, and Profit Margin.
- **Geographic Analysis:** Interactive map showing sales distribution.
- **Trend Analysis:** Monthly order volume tracking.
- **Category Insights:** Breakdown of units sold by category.
 ## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation (Power Query)
